Launching Instructions
1. Download the zip file and extract it
2. Import the zip folder as a new project in Eclipse
3. Download, install, and run Mongo on localhost at port 27017 (the defaults)
3. Run the server in Eclipse
4. Use Chrome to navigate to http://localhost:8080/COMP4601SDA/rest/sda
5. Create a document
6. Examine the other rest calls to be made in the classes "Main.java", and "SDADocumentActions.java". Test accordingly
7. Use the terminal/command prompt to test Mongo is being updated correctly
